# Участие в разработке

Создавайте отдельную ветку и pull request. PHP-код оформляется с отступом в два
пробела, длиной строки до 120 символов и завершающим переводом строки.

Перед отправкой изменений выполните:

```bash
composer validate --strict --no-check-publish
php -l path/to/changed-file.php
pwsh tools/build-marketplace.ps1
```

Отправляя изменение, вы соглашаетесь распространять его по лицензии MIT и
сохранять уведомление об авторских правах разработчика.
