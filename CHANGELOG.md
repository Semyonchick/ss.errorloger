# История изменений

## 1.1.0 — 2026-07-22

- пункт «Ошибки PHP» корректно возвращается в раздел SmartSam административного меню после установки модуля;
- пункт встраивается в общий список SmartSam без отдельной группы и работает с резервным размещением в «Сервисах».
- администратор может скачать исходный файл журнала со страницы модуля.
- улучшены отступы в блоке временной ссылки, ссылка копируется при клике по полю.

## 1.0.2 — 2026-07-22

- исправлена кроссплатформенная сборка `.last_version.zip` в GitHub Actions.

## 1.0.1 — 2026-07-22

- уточнена стандартная установка модуля без Sprint Migration;
- добавлены MIT-лицензия, данные партнёра и подготовка Bitrix Marketplace;
- добавлены CODEOWNERS, проверки GitHub Actions и политика безопасности.

## 1.0.0 — 2026-07-22

- журнал определяется через `exception_handling` в настройках Bitrix;
- одинаковые ошибки объединяются с количеством, первым и последним временем;
- добавлены поиск, карточки деталей и список последних срабатываний;
- добавлена отзывная ссылка без авторизации сроком на 72 часа;
- добавлены Composer-установка и сборка для Bitrix Marketplace.
